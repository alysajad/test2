Evaluation Metrics

The models were evaluated using comprehensive metrics to ensure balanced performance:

  • Accuracy: Overall correctness of predictions
  • Precision: True positive rate among positive predictions
  • Recall: True positive rate among actual positives
  • F1-Score: Harmonic mean of precision and recall
  • AUC-ROC: Area under the receiver operating characteristic curve

Technical Challenges & Solutions

The File Transfer Odyssey

Challenge: VM network configuration prevented simple file transfers

Solution: Implemented secure copy protocol (SCP) after troubleshooting network settings

# Successful SCP command scp "C:\Users\alisa\Downloads\ddos_model.pkl" kali@10.0.2.15:/home/kali/ddos/ # Prerequisites: # 1. Find VM IP: ip a # 2. Enable SSH: sudo systemctl start ssh # 3. Consider firewall rules

Dependency Hell: Externally-Managed Environment

Challenge: Modern Linux distributions block direct pip installations to system Python

Root Cause: Protection mechanism to prevent system instability

Solution: Proper virtual environment implementation

# The correct approach python3 -m venv venv source venv/bin/activate pip install scikit-learn joblib
